Portfolio Website

This repository contains the source code for my personal developer portfolio. The site serves as a central place to showcase my work, provide live demos of my projects, and outline my experience as a full-stack developer.

Purpose

The goal of this portfolio is to present my technical abilities in a clear and organized way. It highlights selected projects, summarizes my development background, and provides links to deployed applications and their respective repositories.

Technology Used:

HTML, CSS, JavaScript (core structure and styling)

Responsive layout for desktop and mobile

GitHub Pages

Features:

Clean, minimalist layout focused on readability

Sections for project showcases, skills, and contact information

Links to live deployments and code repositories

Optimized assets and simple navigation for fast loading

Deployment

The portfolio is deployed using GitHub Pages.
Any changes pushed to the main branch are automatically reflected in the live version.

Contact

For collaboration or inquiries, contact me through the links provided on the site.
